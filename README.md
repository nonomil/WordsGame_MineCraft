# ? Minecraft 单词学习游戏

专为3-6岁儿童设计的英语学习游戏，包含571个精选单词，支持中英文发音、奖励系统和动画效果。

## ? 快速启动

### 方法一：使用启动器（推荐）

#### Windows 用户
1. **双击 `启动游戏.bat`** - 简单易用的批处理启动器
2. **右键 `启动游戏.ps1` → 使用PowerShell运行** - 功能更强大的PowerShell启动器

#### 网页启动器
- **双击 `? Minecraft单词学习游戏.html`** - 带图标的网页启动器，可以检测服务器状态

### 方法二：手动启动
1. 打开命令提示符或PowerShell
2. 进入游戏目录：`cd "游戏文件夹路径"`
3. 运行：`python -m http.server 8000`
4. 在浏览器中访问：`http://localhost:8000`

## ? 游戏特色

### ? 学习模式
- **分组学习**：每10个单词为一组，循序渐进
- **互动选择**：看图选择正确的中文意思
- **即时反馈**：答对获得星星动画和钻石奖励
- **发音练习**：支持中英文发音切换

### ? 测试模式
- **随机测试**：从词库中随机选择题目
- **进度跟踪**：实时显示答题进度和正确率
- **奖励系统**：答对题目获得钻石，集齐10颗钻石获得钻石剑
- **双语发音**：支持中英文发音切换

### ? 奖励系统
- **钻石奖励**：每答对一题获得一颗钻石
- **钻石剑成就**：每获得10颗钻石自动获得一把钻石剑
- **星星动画**：答对题目时的视觉奖励
- **实时统计**：页面顶部显示总钻石和钻石剑数量

## ? 词库信息

- **总词汇量**：571个精选单词
- **分类覆盖**：基础词汇、自然动物、学习用品等
- **难度适宜**：专为幼儿园儿童设计

## ?? 故障排除

### 常见问题

#### 1. CORS错误
**问题**：浏览器显示跨域错误
**解决**：不要直接双击打开`index.html`，请使用启动器或通过HTTP服务器访问

#### 2. 端口被占用
**问题**：8000端口已被其他程序使用
**解决**：启动器会自动尝试8001端口，或手动指定其他端口

#### 3. Python未找到
**问题**：系统提示找不到Python
**解决**：请安装Python并确保添加到系统PATH

#### 4. 词库加载失败
**问题**：游戏显示词库加载错误
**解决**：确保通过HTTP服务器访问，而不是直接打开HTML文件

### 获取帮助
如果遇到其他问题，请检查：
1. Python是否正确安装
2. 是否通过HTTP服务器访问游戏
3. 浏览器控制台是否有错误信息
